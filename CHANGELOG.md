# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2026-04-02

### Added
- **Internationalization (i18n):** Full support for English (`en-us`), Portuguese (`pt-br`), and Spanish (`es`) dynamically generated menus.
- **TV Series Support:** Added a "Series Marathons" catalog with 5 major universes (Dragon Ball, Star Wars, The Walking Dead, Game of Thrones, The Boys).
- **Automated Data Pipeline:** The backend now loads `.json` files from `data/sagas/` generated by a GitHub Action that uses the TMDB API to keep movie IDs up to date.
- **Improved UI:** Added a sleek interactive `/configure` page built with pure HTML/CSS and glassmorphism UI. Copied directly to clipboard and opened seamlessly.
- **Environment Management:** Local `.env` support through `python-dotenv`.
- **Modular Controllers:** Routing is now properly separated into logic blueprints (`controllers/catalog.py`, `controllers/meta.py`).
- **Make Commands:** Added a unified `Makefile` for installing dependencies, running local dev server, testing, and syncing data.

### Changed
- Refactored `app.py`: Reduced size by 50% by extracting out HTML templates to `templates/configure.html` and extracting routes to Blueprints.
- Refactored duplicate routing blocks via single `build_metas` helper method.
- Re-organized Stremio endpoints to use unified Flask URL decorators `defaults={"config_b64": None}` to remove duplicated wrapper functions.
- Stremio add-on `id` bumped to `br.cinemaratona.addon`.

### Fixed
- Fixed bug where `catalog_series` translations were hardcoded improperly on Stremio UI.
- Fixed 20+ missing or incorrect IMDb IDs (007, X-Men, Friday the 13th, Christopher Nolan) across `catalogs.py`.

## [1.0.0] - Initial Release
- Basic implementation of Stremio add-on with static movie catalogs.
- Simple python-based list logic.
