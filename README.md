# 🎬 CineMaratona BR

> **Encontre a ordem certa. Assista do jeito certo.**

O **CineMaratona** organiza sagas, franquias e universos cinematográficos na ordem correta de assistir — seja cronológica ou por data de lançamento. Feito por fãs, para fãs, eliminando a confusão de ordens complexas e garantindo a melhor experiência sem spoilers.

---

### ✨ O que este Addon oferece

| Catálogo | Descrição |
|----------|-----------|
| 🎬 **Sagas de Filmes** | MCU, Star Wars, Harry Potter, 007, John Wick e as maiores franquias da história. |
| 📺 **Sagas de Séries** | Universos expandidos e séries com arcos contínuos (Breaking Bad, Game of Thrones, The Boys). |
| ✨ **Sagas de Animações** | Curadoria completa Disney, Pixar e arcos de Anime organizados. |
| ⭐ **Coleções Especiais** | Filmografias de diretores (Nolan, Tarantino), Ordem Cronológica e Maratonas Temáticas. |

### 🌐 Idiomas suportados
Português (BR) · English · Español

### ⚡ Instalação
1. Acesse a [Página de Configuração](https://cinemaratona-br.vercel.app/configure)
2. Escolha seu idioma e selecione as categorias que deseja ver no Stremio.
3. Clique em **Gerar Link de Instalação**.
4. Instale o link gerado diretamente no Stremio.
5. Bom filme! 🍿

---

### 🛠️ Desenvolvimento e Contribuição

Este addon é construído em **Python (Flask)** e roda em infraestrutura **Serverless (Vercel)**.

- **Modular:** Dados organizados em `data/catalogs/`.
- **Performático:** Cache in-process e Edge Caching configurados.
- **Seguro:** Testes automatizados de endpoint e integridade de dados.

```bash
# Rodar localmente
make install
make dev

# Rodar testes
make test
```
